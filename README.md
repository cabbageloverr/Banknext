# Banknext
**How to run**
1.ติดตั้ง node_module ให้แก่ server และ  client(open terminal ทั้งสองโฟลเดอร์) ด้วยคำสั่ง
```
npm install
```

2.run server ด้วยคำสั่ง
```
npm run dev
```

3.run client ด้วยคำสั่ง
```
npm start
```
